最小调整代价和

发表于 2018-03-05 | 分类于 刷题

给一个整数数组，调整每个数的大小，使得相邻的两个数的差不大于一个给定的整数target，调整每个数的代价为调整前后的差的绝对值，求调整代价之和最小是多少。

注意：可以假设数组中每个整数都是正整数，且小于等于100

分析：
动态规划思想，dp[i][j]代表将第i个数调整为j此时的调整代价，dp[i][j] = min(dp[i-1][k]) + abs(i-j),其中k的范围由target确定，即保证相邻两个数的差值小于等于target。
