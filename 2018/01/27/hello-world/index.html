Hello World

发表于 2018-01-27 | 分类于 其他

个人博客的Hello World，之前偶尔会在其他地方写一写，一直以来都想有自己的博客，搞了一番发现，哇咔咔，原来如此便捷，真的是技术限制了我的想象力。写博客能让思维逻辑更加清晰，当然更重要的是能够向技术大牛靠近一些，希望自己能在不久的将来成为一个有技术的人。那第一篇就记录下搭建博客的历程，初来乍到，有理解不到位的地方，还请不吝赐教…

搭建博客用到了

  • Github 不用多说，程序猿都知道的代码托管网站，用到了其中的page。
  • hexo 一款基于Node.js，能快速生成静态网页的工具。
  • NexT hexo的一个主题，时下挺火的一个主题，简洁美观。
  • Markdown 非常简单的排版工具。

首先需要准备Github账号，并新建仓库，仓库名称为“你的用户名.github.io”，这是固定格式。
然后安装依赖工具Node.js和Git。
